3. Escolha a opção "Redefinir este PC" e siga as instruções para redefinir o PIN.

Ne vous inquiétez pas, c'est un problème courant. Voici quelques méthodes pour vous aider à retrouver l'accès à votre ordinateur portable

Méthode 1 : Essayez le code PIN par défaut



Si vous n'avez pas modifié le code PIN par défaut, vous pouvez essayer en utilisant le code par défaut « 0000 » ou « 1234 ». Si cela ne fonctionne pas, passez à la méthode suivante.

Méthode 2 : effectuer une restauration du système



Si vous avez créé un système point de restauration, vous pouvez essayer de restaurer votre système à une date antérieure à laquelle le code PIN fonctionnait encore. Pour ce faire

1. Redémarrez votre ordinateur portable et passez en mode de récupération (généralement en appuyant simultanément sur le bouton d'alimentation et sur le bouton de réduction du volume).
2. Suivez les instructions à l'écran pour sélectionner l'option de restauration du système.
3. Choisissez un point de restauration à partir d'une date antérieure à laquelle le code PIN fonctionnait.

Méthode 3 : utilisez le mode de récupération LG



Si les méthodes ci-dessus ne fonctionne pas, vous pouvez essayer d'utiliser le mode de récupération LG. Pour ce faire

1. Redémarrez votre ordinateur portable et passez en mode de récupération (généralement en appuyant simultanément sur le bouton d'alimentation et sur le bouton de réduction du volume).
2. Suivez les instructions à l'écran pour sélectionner l'option de récupération.
3. Choisissez l'option "Réinitialiser ce PC" et suivez les instructions.

Méthode 4 : Réinitialiser le code PIN



Si aucune des méthodes ci-dessus ne fonctionne vous pouvez essayer de réinitialiser le code PIN. Veuillez noter que cela effacera toutes les données de votre ordinateur portable, utilisez donc cette méthode uniquement si vous souhaitez réinitialiser le code PIN.

1. Redémarrez votre ordinateur portable et passez en mode de récupération (généralement en appuyant simultanément sur le bouton d'alimentation et sur le bouton de réduction du volume).
2. Suivez les instructions à l'écran pour sélectionner l'option de récupération.
3. Choisissez l'option « Réinitialiser ce PC » et suivez les instructions pour réinitialiser le code PIN.

Méthode 5 : Visitez un centre de service LG agréé



Si aucune des méthodes ci-dessus ne fonctionne, vous devrez peut-être vous rendre dans un centre de service LG agréé pour obtenir une assistance supplémentaire. Ils peuvent vous aider à réinitialiser le code PIN ou à récupérer votre ordinateur portable.
